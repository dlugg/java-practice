# Rock, Paper, Scissors (Java)

A simple Java implementation of the classic "Rock, Paper, Scissors" game.

## Features
- Play against the computer.
- First to reach 3 points wins.
- Score is displayed after each round.

## How to Run
1. Ensure you have Java (JDK) installed.
2. Compile the file: `javac src/Main.java`
3. Run the program: `java -cp src Main` (or use your IDE).

## Controls
- `1` — Rock
- `2` — Paper
- `3` — Scissors
